# FloppyBird  
2015年春学期「スマートデバイスプログラミング」最終課題  

君は1枚のフロッピーディスクを渡された。  
無事に運べるかは君の指にかかっている。大事なデータを磁気から守れ！  


## 操作方法  
ディスプレイのタッチのみ  
## ルール  
天井、床、飛んでくる棒磁石に当たるとゲーム終了  
## 動作確認端末  
Nexus 5, Nexus 7 (いずれも Android 5.1.1)  
解像度 1920 * 1080 を想定して作っています
